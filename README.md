# AWS Static Website Hosting Project 🌐

## 📌 Project Overview

This project demonstrates how to host a static website using AWS S3 Free Tier.

The website displays a simple welcome message and is publicly accessible via S3 static website endpoint.

## 🚀 Technologies Used

* AWS S3
* HTML
* GitHub

## ⚙️ Steps Performed

1. Created S3 bucket
2. Enabled Static Website Hosting
3. Uploaded index.html file
4. Configured bucket policy for public access
5. Accessed website using S3 endpoint URL
